## 核心讲义

### 发一声,听回声,数时间

HC-SR04 超声波测距的原理和蝙蝠、声呐一样:**发出一束超声波,等它撞到障碍物弹回来,测量往返用了多久**。声音在空气中速度基本恒定(约 **343 m/s**,即 0.0343 cm/µs),知道时间就能算出距离。

因为声波是「去 + 回」跑了两倍距离,所以:

> **距离 = 声速 × 回声时间 ÷ 2**

### 触发与回响:两个引脚的对话

模块有四个脚:`VCC`、`GND`、`Trig`(触发)、`Echo`(回响)。测一次距离的时序是:

1. **触发:** 单片机在 `Trig` 上给一个 **10 µs** 的高电平脉冲,命令模块「发声」。
2. **发声:** 模块自动发出 8 个 40 kHz 的超声脉冲。
3. **回响:** 模块把 `Echo` 拉高,高电平持续的时间**正好等于**声波往返的时间;收到回声后拉低。

所以我们要做的,就是「测量 Echo 高电平持续了多少微秒」。

### pulseIn 与距离公式

Arduino 的 `pulseIn` 正好用来测一个脉冲的宽度(µs):

```cpp
const int trig = 9, echo = 10;

void setup() {
  pinMode(trig, OUTPUT);
  pinMode(echo, INPUT);
  Serial.begin(9600);
}

void loop() {
  // 发 10µs 触发脉冲
  digitalWrite(trig, LOW);  delayMicroseconds(2);
  digitalWrite(trig, HIGH); delayMicroseconds(10);
  digitalWrite(trig, LOW);

  long us = pulseIn(echo, HIGH);       // 回声脉宽(微秒)
  float cm = us * 0.0343 / 2.0;        // 换算成厘米

  Serial.print(cm); Serial.println(" cm");
  delay(100);
}
```

上方交互图拖动距离,可看到回声时间随距离线性增长——50 cm 对应约 2915 µs。

- **盲区:** 太近(< 2 cm)时回声还没测就已返回,读数不准。
- **角度:** 超声波是一个约 15° 的锥,面对倾斜或吸音的表面(布料、海绵)回声弱,可能测不到。
- **温度:** 声速随温度变化,高精度场合要按温度修正。
- **两次测量间留间隔**(≥ 60 ms),让上一束回声散尽,否则会互相干扰。

> ✦ **要点:** 这是一把「声学卷尺」。全部秘密就是:触发一下 → 用 `pulseIn` 量回声时间 → 乘声速除以 2。理解了「时间差 = 距离」,你就理解了从倒车雷达到激光测距的一大类传感器。
