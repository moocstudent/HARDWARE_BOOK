## 核心讲义

> ✦ 这个项目换了个视角:前面的模块都在**自己造硬件**,这一模块学的是如何**评估并驯服一台成品设备**——以 R36S 这类平价 Linux 掌上游戏机为例。会看规格、会刷系统、会对接程序,你就从「用户」变成了「能改能编的开发者」。

### 别被「预装 15000 款游戏」骗了

掌机的商品页最爱写「内置上万款游戏」「支持 21 种模拟器」。但这些数字**几乎不说明性能**——上万款游戏只是一张塞满 ROM 的 SD 卡,任何机器都能「预装」,可它跑不跑得动是另一回事。真正决定体验的,是两个营销词里不会强调的东西:

1. **这台机器的 SoC,能流畅模拟到哪一代主机?**
2. **屏幕、电池、按键手感、做工,能不能长期用得住?**

学会看这两点,你就不会为「游戏数量」这种虚指标买单,而能判断一台机器到底值不值、适不适合你想玩的东西。

### SoC 是心脏:以 RK3326 为例

**SoC(System-on-Chip,片上系统)**把 CPU、GPU、内存控制器、各种外设集成在一颗芯片里,是整台掌机的心脏——**它几乎决定了一切性能上限**。R36S 用的是瑞芯微 **RK3326**:

逐项读懂这张表:

- **CPU:四核 ARM Cortex-A35**,64 位,最高约 1.5 GHz。关键在 **A35 是一颗「能效核」**——ARM 的核分两类:追求性能的大核(A7x、A5x)和追求省电的小核(A35、A53、A55)。A35 属最省电那档,好处是发热小、续航长,代价是**单核算力有限**。而大多数模拟器是**单线程吃 CPU** 的(一个游戏主要压在一个核上),所以「四核」并不能让老游戏跑得更快——真正决定的是**单核性能**,而 A35 的单核性能不高。这直接框定了模拟能力的上限。
- **GPU:Mali-G31**,入门级,足够跑 2D 和早期 3D 游戏,但 PS2、GameCube 那种重 3D 就力不从心。
- **内存:约 1 GB LPDDR**,跑模拟器和 Linux 前端够用,但也谈不上宽裕。

**读规格的核心心法:别只看核数和主频,要看架构(大核还是小核)和单核性能。** 一颗 1.5 GHz 的 A35,单核性能远不如一颗 1.8 GHz 的 A55 或 A76。

### 性能 vs 模拟世代:能玩到哪一代

模拟一台主机有多难,取决于**被模拟主机的复杂度**——越晚的主机架构越复杂、越吃算力。而且模拟不是「原速就行」:模拟器要在软件里逐条翻译目标 CPU 的指令,通常需要**数倍于原机的算力**才能跑到原速。这就是为什么一台现代掌机能轻松模拟 30 年前的红白机,却对十几年前的 PS2 束手无策。

一个粗略的对应关系(拖动上方交互图的「SoC 档次」滑块查看不同档位):

| 主机世代 | 入门档(RK3326 / R36S) |
|---|---|
| NES / SNES / GB / GBC / MD(世嘉) | ✅ 完全流畅 |
| GBA / PC Engine | ✅ 完全流畅 |
| PS1 | ✅ 基本流畅(选对内核) |
| N64 / PSP / NDS | ⚠️ 视游戏而定,部分吃力 |
| 世嘉土星 / DreamCast | ⚠️ 吃力,很多跑不好 |
| PS2 / GameCube / Wii | ❌ 跑不动 |
| Switch / PS3 及以上 | ❌ 完全不行 |

**结论:** R36S 这类入门机,是「**PS1 及更早的黄金年代**」的绝佳性价比之选——红白机、SFC、GBA、街机、初代 PS 的海量经典都能顺畅玩。想稳玩 PSP / N64 / DS,要看**中端机**(如 RK3566、H700 芯片);想碰 PS2 / GameCube,则需要**高端 SoC**(RK3588、骁龙等),价格和体积也随之上升。**先想清楚你主要想玩哪一代,再据此选 SoC 档次**,别为用不上的性能多花钱,也别买了才发现玩不了心心念念的游戏。

### 屏幕、电池、手感与做工

规格表之外,这些「体感」项决定你会不会真的一直用它——它们在参数页上往往被一笔带过,却最影响长期体验:

- **屏幕**:R36S 是 **3.5 英寸 IPS,640×480**,全贴合(OCA)。要看三点:**分辨率**(决定清晰度,老游戏本身分辨率低,640×480 够用)、**面板类型**(IPS 色彩正、可视角广;TN 便宜但偏色、斜看变色)、**是否全贴合**(全贴合的屏玻璃与显示层之间无空气层,通透感更好、不易进灰)。
- **电池**:R36S 是 **3500 mAh,约 6 小时**续航,充电约 2 小时。续航是**电池容量与 SoC 功耗的平衡**——A35 省电,所以中等容量也能撑较久。看续航别只看毫安时,要结合芯片功耗一起看。
- **手感**:十字键(D-pad)的段落感、摇杆的精度与回中、肩键(尤其是模拟扳机 vs 数字按键)的布局,是长时间游玩舒适度的关键。这些**规格表上完全看不到**,只能看实机评测和上手体验。廉价机常在按键手感上妥协。
- **散热与做工**:塑料件的公差、按键有没有旷量、长时间运行发不发烫、外壳缝隙是否整齐——平价机的成本压力常体现在这些细节上。

### 避坑:克隆机与虚标

平价掌机市场山寨、贴牌、虚标横行,采购时要格外当心:

- **同壳不同芯**:市面上有大量「长得一模一样」的机器,里面可能是完全不同、性能差很多的芯片。**认准 SoC 型号**,而不是外形或卖家宣传。
- **虚标配置**:标称的内存、电池容量、屏幕素质可能注水。参考第三方评测拆解,而非商品页。
- **「Wi-Fi」要另配**:很多入门机(包括 R36S)**没有内置 Wi-Fi**,联网要额外买一个 USB-C 无线适配器(见 GC4)。别想当然。
- **看社区**:一个健康、活跃的固件社区(能刷 ArkOS / JELOS 等第三方系统,见 GC2)是机器靠谱、值得买的强信号——社区愿意为它做系统,说明硬件规范、用户基数大、坑已被踩平。

### 内存与存储:容易被忽视的两块

除了 SoC,还有两个常被一笔带过、却影响体验的部件:

- **内存(RAM)**:模拟器把游戏、状态、缓冲都放在内存里。约 1 GB 对 PS1 及更早绰绰有余,但跑更重的模拟器(N64/PSP)或开大缓冲、着色器时会吃紧。内存不够会导致卡顿甚至崩溃。它不像 SoC 那样决定「能不能跑」,但决定「跑得顺不顺」。
- **存储(SD 卡)**:游戏从 SD 卡加载。**卡的速度等级**(Class 10、U1/U3、A1/A2)影响游戏加载时间和大型光盘游戏(PS1 的 .chd)的读取流畅度。买一张正品、够快的卡,比省那几块钱重要——山寨扩容卡是掌机社区最大的坑之一(标称 256 GB 实则 32 GB,存满就丢数据)。

### 屏幕比例与整数缩放

老游戏大多是 **4:3** 的比例(甚至更方)。R36S 的 640×480 正是 4:3,和 SNES、PS1 等原生比例吻合——这不是巧合,而是为复古而选。如果买一台 16:9 宽屏掌机玩 4:3 老游戏,要么两侧留黑边,要么强行拉伸变形。

进阶一点还有**整数缩放(integer scaling)**的概念:当屏幕分辨率是游戏原生分辨率的整数倍时,每个原始像素恰好放大成 n×n 个屏幕像素,画面锐利无模糊;非整数倍则需要插值,画面发糊或出现不均匀的像素。追求像素完美的玩家会特别看重这一点。理解它,你就明白为什么「屏幕分辨率」不是越高越好,而是要和你想玩的世代「对得上」。

### 形态、人体工学与散热

- **横版 vs 竖版**:横版(如 R36S,像小号 PSP)适合动作、格斗、模拟摇杆的游戏;竖版(Game Boy 造型)适合 NES/GB 这类经典掌机游戏、更便携。选哪种看你主玩什么、怎么带。
- **重量与握持**:太重手累,太轻廉价感强,握持曲面、按键位置决定长时间舒适度。
- **散热**:入门 SoC 功耗低,通常被动散热(靠外壳)就够;但玩重度模拟长时间高负载时会发热,可能触发**降频(thermal throttling)**——芯片为降温自动降速,表现为「玩一会儿就开始卡」。做工好的机器散热设计更合理。

### 声音与其它

规格页常忽略的还有**音频**:R36S 标称「8W 腔体扬声器 + 支持 aux(耳机)」。外放音质、有没有耳机孔、蓝牙音频延迟,对体验都有影响。此外**震动、按键背光、Type-C 供电、能否边充边玩**等小细节,也值得按需求核对。

### 认识整个掌机江湖

R36S 只是庞大平价 Linux 掌机生态里的一员。了解大致格局,有助于横向比较、按需选择:

| 芯片档 | 代表机型(举例) | 大致能玩到 |
|---|---|---|
| 入门(RK3326) | R36S、RG351 系列 | PS1 及更早 |
| 中端(RK3566 / H700) | RG353、RG40xx、Anbernic 中端 | + PSP / N64 / DS |
| 高端(RK3588 / 骁龙) | Retroid Pocket 高端、AYN Odin 等 | + PS2 / GC / 部分 Switch |

品牌上,**Anbernic、Miyoo、Powkiddy、Retroid** 等是常见玩家。不同机型芯片可能相同(性能相近)、外形与做工各异。**先按「想玩到哪一代」锁定芯片档,再在该档里比外形、屏幕、做工、社区支持挑具体型号**——这就是把本章所有要点串起来的选购方法。

### 一套可照做的选购决策流程

把本章要点串成一个可执行的步骤,下次买机器照着走:

1. **先定「玩到哪一代」**:诚实问自己主要想玩什么。只玩 FC/SFC/GBA/街机/PS1?入门档足矣,别多花钱。想稳玩 PSP/N64/DS?中端。要碰 PS2/GC?高端。
2. **据此锁定 SoC 档**:用上文那张「芯片档 → 能玩到哪」的表,圈定一个档位(如入门 = RK3326)。
3. **在该档里查具体机型的社区支持**:能刷 ArkOS/JELOS、社区活跃的优先(靠谱信号)。
4. **比体感项**:横版还是竖版?屏幕比例(4:3 复古 / 16:9)?按键手感评测口碑?电池够不够?
5. **核对隐藏成本**:要不要另配 Wi-Fi 适配器?附带的 SD 卡够不够快、是不是正品?
6. **认准 SoC 型号下单**:别被「上万游戏」带偏,确认芯片型号与规格,选口碑好的卖家。

走完这六步,你买到的就不是「一台塞满游戏的盒子」,而是「一台匹配你需求、你了解其能力边界、还能深度折腾的小电脑」。

### 常见疑问

| 疑问 | 解答 |
|---|---|
| 核越多越强吗? | 不。模拟多为单线程,**单核性能**才关键;四核 A35 不如双核大核 |
| 屏幕分辨率越高越好吗? | 不。要和游戏世代匹配,4:3、整数缩放对复古更重要 |
| 「预装上万游戏」值钱吗? | 那只是一张 SD 卡的内容,不代表性能,别为它溢价 |
| 便宜机能玩 PS2 吗? | 入门档不行,PS2 需要高端 SoC(见能力表) |
| 没内置 Wi-Fi 是坑吗? | 是常见情况,提前确认要不要另配 USB-C 适配器 |
| 怎么防买到山寨/虚标? | 认 SoC 型号、看第三方拆解评测、选活跃社区支持的机型 |

> ✦ **要点:** 选掌机,**第一看 SoC**——它决定「能模拟到哪一代」,而看 SoC 要看**架构与单核性能**,不是核数和主频。R36S(RK3326,四核 A35)是 PS1 时代的性价比之选。**第二看屏幕、电池、手感、做工**这些体感项,它们不在参数页却最影响长期体验。游戏数量是营销,SoC 档次、做工与社区支持才是真相。先想清楚要玩哪一代,再对号入座地选。
