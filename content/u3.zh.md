## 核心讲义

### RP2040:树莓派自研的双核芯片

树莓派 Pico 用的是 **RP2040**——一颗**双核 ARM Cortex-M0+**,主频 133 MHz,3.3V 逻辑。它性价比高、生态好,而且有一个别家没有的杀手锏:**PIO**。

### PIO:用状态机「造」外设

普通单片机的外设(UART、SPI 等)是芯片出厂就焊死的,有几个是几个。RP2040 的 **PIO(可编程 I/O)**给了你 8 个小小的**状态机**,可以用很简单的指令编程,让它们**在引脚上精确地生成/采集任意时序**。

这意味着:如果你需要一个芯片本没有的接口(比如驱动 WS2812 灯带那种苛刻时序、或者多路额外的串口),不必占用 CPU 死等,交给 PIO 精准地跑就行。这是 RP2040 最独特、最值得体会的设计。

### 两种开发语言

Pico 支持两条路,按喜好选:

- **C/C++(官方 SDK):** 性能最好、最贴近硬件,也可用 Arduino 框架。
- **MicroPython:** 在单片机上跑精简版 Python,交互式、上手极快,非常适合教学和快速原型。

MicroPython 点灯:

```python
from machine import Pin
from time import sleep
led = Pin(25, Pin.OUT)     # 板载 LED
while True:
    led.toggle()
    sleep(0.5)
```

### 引脚与供电要点

Pico 有 26 个可用 GPIO、3 个 ADC 输入(12 位),3.3V 逻辑。供电可走 USB 的 `VBUS`(5V),板载稳压给出 `3V3`。和 ESP32 一样,**引脚不耐 5V**,接 5V 器件要电平转换。

> ✦ **要点:** RP2040 是一颗均衡好用的双核 M0+,亮点在 **PIO**——把「时序敏感的外设」从 CPU 卸载到可编程状态机。开发可选 C/C++ 或 MicroPython 两条路。牢记 3.3V 逻辑。
