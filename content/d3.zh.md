## 核心讲义

### 数字脚只能开/关,怎么调出「一半亮」?

一个输出引脚只有 HIGH 和 LOW 两个电平,似乎只能让 LED 全亮或全灭。但如果我们让它**飞快地开关**,并控制「开的时间占多大比例」,人眼和电机来不及跟上这种高速切换,只会感受到一个**平均值**。这就是 **PWM(Pulse-Width Modulation,脉宽调制)**。

### 占空比决定平均电压

一个周期里,引脚为 HIGH 的时间占比叫**占空比(duty cycle)**:

- 占空比 0% → 一直 LOW → 平均 0V → 灯灭
- 占空比 50% → 半开半关 → 平均约 Vcc/2 → 半亮
- 占空比 100% → 一直 HIGH → 平均 Vcc → 全亮

> **平均电压 ≈ 占空比 × Vcc**

上方交互图拖动占空比,能看到方波的高电平变宽、平均线(虚线)随之上移、色块变亮。

### analogWrite:0–255 的刻度

在 Arduino 上,PWM 用 `analogWrite` 输出,取值 **0–255**(8 位):

```cpp
analogWrite(9, 0);     // 占空比 0%   全灭
analogWrite(9, 64);    // 约 25%      偏暗
analogWrite(9, 128);   // 约 50%      半亮
analogWrite(9, 255);   // 100%        全亮
```

呼吸灯示例:

```cpp
void setup() { pinMode(9, OUTPUT); }
void loop() {
  for (int v = 0; v <= 255; v++) { analogWrite(9, v); delay(4); }
  for (int v = 255; v >= 0; v--) { analogWrite(9, v); delay(4); }
}
```

### 不是每个脚都能 PWM

`analogWrite` 只在**特定引脚**上是真正的硬件 PWM。Arduino Uno 上是标注了 `~` 的脚:**3、5、6、9、10、11**。其他脚调用 `analogWrite` 得不到平滑的调光效果。

### 频率、抖动与平滑

PWM 的**频率**(每秒开关多少次)也很重要。频率太低,LED 会看到闪烁、电机会抖动或啸叫。Uno 默认约 490 Hz 或 980 Hz,调光足够。若要得到真正平滑的**模拟电压**(比如驱动运放),可在 PWM 输出后接一个 RC 低通滤波(见 CF4)把方波「抹平」成直流。

> ✦ **要点:** PWM 不是真的模拟输出,而是用「快速开关 + 占空比」在时间上平均出一个等效电压。调光、调速、生成模拟电压——都是它。记住三件事:占空比、可用引脚、频率。
