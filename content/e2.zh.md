## 核心讲义

### LED 不是电阻

发光二极管(LED)是一种**二极管**:电流只能从正极(阳极,长脚)流向负极(阴极,短脚/平边),反着几乎不导通。更关键的是,它的电压-电流关系**不是**一条直线,而是一道「门槛」:

- 电压低于某个值时,几乎不导通、不发光;
- 一旦超过这个门槛——**正向压降 Vf**——电流会**急剧上升**,而它两端电压几乎钉在 Vf 不再涨。

| LED 颜色 | 典型 Vf | 典型工作电流 |
|---|---|---|
| 红 | ~1.8–2.0 V | 5–20 mA |
| 黄 / 橙 | ~2.0–2.2 V | 5–20 mA |
| 绿 | ~2.0–3.0 V | 5–20 mA |
| 蓝 / 白 | ~3.0–3.4 V | 5–20 mA |

因为电流对电压极其敏感,**你不能直接控制电流**。如果把 5V 直接接到一颗 Vf=2V 的红色 LED 上,理论上要流过无穷大的电流——实际就是瞬间烧毁。

### 限流电阻:一颗电阻救一颗灯

解决办法是串联一颗**限流电阻**,把「多余的电压」吃掉,从而把电流限制在安全范围。电源电压 Vs 一部分落在 LED(Vf),剩下的落在电阻上:

> 电阻两端电压 = Vs − Vf  
> 于是 **R = (Vs − Vf) / I**

举例:5V 电源,红色 LED(Vf=2V),想要 10 mA:

```text
R = (5 − 2) / 0.010 = 300 Ω
```

买不到正好 300Ω?选更大的标准值(如 330Ω)——电流会略小、灯略暗,但**绝对安全**。宁大勿小。

### 极性别接反

- **长脚 = 阳极(+)**,接向电源正 / 单片机引脚一侧;
- **短脚 / 外壳平边 = 阴极(−)**,接向地(GND)。

接反了 LED 不会亮(但通常不会损坏,普通 LED 反向耐压足够)。

### 用 Arduino 点亮它

把「引脚 → 电阻 → LED → GND」接好,代码就是最经典的 Blink:

```cpp
void setup() {
  pinMode(13, OUTPUT);      // 13 号脚设为输出
}
void loop() {
  digitalWrite(13, HIGH);   // 引脚输出 5V,点亮
  delay(500);
  digitalWrite(13, LOW);    // 引脚输出 0V,熄灭
  delay(500);
}
```

> ⚠ **绝不省略电阻。** 「先接上看看亮不亮」是新手烧 LED 的头号原因。哪怕临时测试,也串一颗 220Ω 或 330Ω。上方交互图里把电阻调小,你会看到电流冲进「过流」红区。
