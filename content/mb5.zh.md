## 核心讲义

### 一旦接触病人,安全压倒一切

前面四章都在讲「怎么测得准」。这一章讲的是**底线**:设备一旦与人体有电气连接,安全就压倒一切,再准的监测也不能以危险为代价。医疗电子最核心、也是与普通电子最不同的一条要求,就是**患者安全(patient safety)**——绝不能有危险电流经病人流过。

为什么这么严?人体是导电的,而心脏对微小电流极其敏感:流经心脏区域**几十微安**的电流,就可能诱发心律失常甚至室颤。普通电子设备漏一点电到外壳,人摸到最多是「麻一下」;但对一个躺在床上、皮肤潮湿、身上可能还插着其它导联的病人,同样的漏电流可能是致命的。因此医疗设备对**漏电流(leakage current)**有极其严格的上限。

### 隔离:切断危险电流的通路

保证患者安全的核心手段是**隔离(isolation)**。「隔离」的含义是:信号和能量可以传过去,但**危险的电流回路被物理切断**——接触病人的那一侧,与市电、与大地之间没有直接的导电通路。实现隔离靠的是「非导电耦合」:

- **变压器**:靠磁场耦合能量,原副边不导通;
- **光耦 / 数字隔离器**:靠光或电容耦合信号,输入输出之间是绝缘的;
- **电容耦合**:靠交变电场传信号。

本项目在几个层面做隔离:

- **隔离电源**:原型阶段可用普通 5 V / 12 V 电源验证功能;但产品阶段必须换成 **IEC 60601-1 医疗级电源**(如明纬 GST / RPS 系列)。医疗电源与普通电源的区别,正在于它的隔离等级和漏电流指标是**专为接触病人设计**的,并有认证背书。
- **隔离放大器 / 数字隔离器**:在「接触病人的采集前端」与「主控」之间插入一道隔离屏障。模拟信号可用 **ISO124** 隔离放大器传过去;数字总线(I²C / SPI)可用 **ADuM 系列数字隔离器**——让信号过、危险电流不过。这样即使前端某处出故障,危险电流也过不到病人身上,反之亦然。
- **电平转换**:3.3 V 与 5 V 器件混用时用电平转换器(见 MC2),避免过压损坏——这虽不直接关乎患者安全,但属于同一类「接口保护」工程。

### 接地、共地与「接地隔离」

初学者常把「隔离」和「共地」搞混。回顾前面模块:很多传感器、总线都要求「共地(common ground)」才能正常通信(如 UART 的共地、I²C 的地参考)。但在医疗设备里,「病人侧」和「市电侧」恰恰**不能**共地——否则市电的地电位波动、漏电流就会经由病人这条路走。

解决之道正是数字隔离器:隔离器**两侧各有独立的地**,信号靠光/电容跨过屏障传递,两个地之间没有直接连接。于是「病人侧」自成一个悬浮的、安全的电气孤岛,内部器件之间照样共地、照样通信;而这个孤岛与外界市电之间被牢牢隔开。理解这一点,就理解了医疗电子布局的精髓。

### 断电也不能中断:备用电池

病人监护是**不能中断**的关键功能:停电了、转运换床了,监测不能就此停摆——恰恰是这些时刻病人更需要照看。所以要加一组**备用电池**,做不间断供电:

- **磷酸铁锂(LiFePO4)电池组**:相比常见的锂聚合物,LiFePO4 **热稳定性更好、更安全、循环寿命更长**,在医疗/工业场景更受青睐(安全本身也是患者安全的一部分)。
- **BMS(电池管理系统)**:负责均衡各电芯、过充/过放/过流/过温保护。没有 BMS 的锂电池组是危险的。
- **无缝切换**:市电正常时充电并供电,市电中断时电池立即接管,系统不重启、不丢数据。

### 多床组网:CAN 总线(可选)

若要把一整个病区多张床的节点串联成一个监护网络,汇总到中央护士站,**CAN 总线**是工业级的首选(回顾 BUS 模块对总线的比较):

- **抗干扰强**:差分信号 + 双绞线,在电气噪声大的医院环境里依然可靠;
- **多节点、长距离**:一条总线挂几十个节点、跑上百米;
- **有仲裁机制**:多个节点同时发消息不会冲突,高优先级(如报警)先走。

用 **SN65HVD230 收发器**(或 **MCP2515** 控制器 + 收发器)把每张床的节点挂上一条 CAN 主干。注意:跨床通信时,总线本身也要考虑隔离,防止一张床的故障经总线影响其它床。

### 工作台:开发这套系统要备什么

这是一次性采购的开发工具,做任何硬件项目都用得上:

- **万用表**:测电压、电流、通断,排障第一工具;
- **台式电源**:可调压限流,安全地给电路供电、做实验;
- **烙铁 / 焊台**:焊接元件与接线;
- **逻辑分析仪(Saleae 兼容款)**:抓 I²C / SPI / UART 的时序波形,调总线通信的利器(回顾 BUS 模块);
- **示波器(或 USB 示波器)**:看模拟波形,调 BCG、看噪声必备;
- **面包板 + 跳线**:快速搭原型;
- **标定砝码**:已知质量,用于校准称重传感器(见 CB2)。

### 从原型到「受监管医疗产品」还差什么

这是最诚实、也最重要的一节。上面的爱好级方案能验证原理、能做出令人惊艳的演示,但它**不是医疗器械**,不能真的用在病人身上。要通过 **IEC 60601-1(通用安全)/ -1-2(电磁兼容)/ -1-8(报警系统)** 等标准测试、成为可上市的合规产品,需要把这些替换/升级:

| 原型级 | → 产品级 |
|---|---|
| 普通 5V/12V 电源 | **IEC 60601-1 医疗电源**、经认证的隔离屏障 |
| HX711 / 面包板前端 | 医疗级**隔离模拟前端** + **ADS1220 / ADS131** 级 ADC |
| DIY Velostat 压力网格 | 商用**校准压力垫**(Tekscan / XSensor) |
| 杜邦线 / 串口 | 工业级 **CAN 总线主干网** + 屏蔽线缆 |
| 面包板裸露电路 | 密封、达 **IP 防护等级**、可清洁的外壳 |
| 无 EMC 考量 | **EMC 屏蔽**与滤波,通过 -1-2 抗扰/辐射测试 |

这些不是「锦上添花」,而是能不能合法用于病人的**分水岭**。合规还涉及风险管理(ISO 14971)、软件生命周期(IEC 62304)、临床验证等一整套工程与法规流程——远超硬件本身。

### 漏电流到底多小才安全

前面说「几十微安就可能有危险」,这不是随口一说,而是标准里有明确数字的。IEC 60601-1 对不同工况下的**漏电流**规定了上限(数量级示意,具体以标准为准):

| 漏电流类型 | 正常工况上限(量级) | 说明 |
|---|---|---|
| 对地漏电流 | 数百 µA | 设备外壳经保护地流走的电流 |
| 对外壳(接触)漏电流 | ~100 µA | 人碰到外壳时可能流经人体 |
| 患者漏电流 | ~10–100 µA | 经与病人相连的部分流过病人 |

注意「患者漏电流」的限值最严——因为它直接经过病人。而且**单一故障工况**(某个元件坏了)下的限值也有规定:哪怕坏了一个元件,漏电流也不能超过危险值。这就是「单一故障安全(single-fault safe)」原则:设备要能容忍任意一个部件失效而不伤人。理解这些数字,你就明白为什么医疗设计要层层隔离、留足余量——不是过度谨慎,而是标准的硬性要求。

### 应用部分的分类:B、BF、CF

医疗设备里,直接或间接接触病人的部分叫**应用部分(applied part)**,按防护等级分三类,越往后越严:

- **B 型**:基本防护,应用部分可能接地。
- **BF 型**(F = Floating 浮地):应用部分是**浮空隔离**的,不与地相连,防护更好。我们这套床的传感器接触部分,至少要做到 BF 级的隔离思路。
- **CF 型**(C = Cardiac 心脏):最高等级,用于可能直接接触心脏的场合(如心内导管),漏电流限值最严。

护理床虽不接触心脏,但只要有电极/传感器贴着病人,就要按「浮地隔离、限制患者漏电流」的思路去做——这正是前面数字隔离器、隔离电源要解决的问题。

### 两道防线:MOOP 与 MOPP

IEC 60601 把「防止触电」的措施叫**保护手段(Means of Protection, MOP)**,并区分两种:

- **MOPP(Means of Patient Protection,患者防护)**:保护病人的隔离屏障,要求最高(更大的爬电距离、更高的绝缘耐压)。
- **MOOP(Means of Operator Protection,操作者防护)**:保护操作者(护士/医生)的措施,要求次之。

标准通常要求**两道独立的防护**(2× MOP)——单靠一道隔离不够,要有第二道作为冗余,任意一道失效还有另一道兜底。这又是「单一故障安全」在结构上的体现。你在选医疗电源、隔离器时,看到「2 MOPP」这样的标注,指的就是它内部提供了两道患者级隔离。

### 医院里的电磁兼容(EMC)

医院是**电磁环境极其复杂**的地方:电刀、核磁、各种大功率设备、无线对讲……一台监护设备既不能被它们干扰而误判/死机(**抗扰度**),也不能自己发出干扰影响别的设备(**发射**)。这就是 **IEC 60601-1-2(电磁兼容)** 要管的事。对策包括:屏蔽线缆与外壳、电源与信号线加滤波、合理接地、PCB 布局讲究……我们的原型用杜邦线在面包板上跑,EMC 基本为零——这也是它「只能演示、不能上临床」的重要原因之一。

### 不止硬件:合规是一整套工程

最后要打破一个常见误解:**把原型变成合规医疗产品,远不止「换几个医疗级器件」。** 它是一整套受法规约束的工程流程:

- **风险管理(ISO 14971)**:系统性地识别每一种可能伤害病人的风险,评估、缓解、留档。
- **软件生命周期(IEC 62304)**:医疗软件的开发、验证、维护都有规范——你那段判断呼吸的算法,在产品里要有需求、设计、测试、追溯的完整链条。
- **可用性工程(IEC 62366)**:界面要防止护士在疲劳、紧急时误操作。
- **临床验证**:证明它在真实病人身上确实有效、安全。
- **质量体系(ISO 13485)**、法规注册(如 NMPA/FDA/CE)……

硬件隔离只是这座大厦的地基。理解这一点,能让你既不轻视原型(它验证了核心原理),也不高估原型(它离产品还隔着整个法规与工程体系)。**「能做出来演示」和「能合法用在病人身上」之间,隔着的正是这一整套东西。**

### 原型阶段的安全底线(即使只是玩)

即便你做的只是桌面原型、不上真病人,也有几条安全底线务必守住——毕竟你自己就是那个「接触设备的人」:

- **原型别接市电裸测**:用现成的、认证过的 USB 电源或电池给低压电路供电,不要自己搭市电到低压的变换。市电致命,不是练手的地方。
- **接触人体的一律隔离/低压**:任何要贴到皮肤的电极,都应工作在安全的低电压、且与任何可能接触市电的部分隔离。
- **电池要带保护**:锂电池必须配 BMS,别用无保护的裸电芯,别短路、别过充。
- **明确「这是玩具」**:原型的数据不能用于任何真实医疗决策。写清楚、贴标签,别让别人误当真设备用。

### 设计自检清单(走向产品时对照)

当有一天真要把它往产品推,这张清单能帮你对照差距(逐项都指向前文某个概念):

- [ ] 接触病人的部分是否**浮地隔离**(BF 思路)?
- [ ] 是否有**两道独立防护**(2× MOP)?
- [ ] 各种工况下**漏电流**是否在限值内,含**单一故障**工况?
- [ ] 电源是否为 **IEC 60601-1 医疗级**?
- [ ] 数字/模拟信号跨隔离屏障是否用了**隔离器**(ADuM/ISO124)?
- [ ] 是否做了 **EMC**(屏蔽、滤波、接地)以应对医院环境?
- [ ] 断电是否有**电池 + BMS** 无缝续航?
- [ ] **报警系统**是否分级、防漏报、能自检(IEC 60601-1-8)?
- [ ] 是否有**风险管理(ISO 14971)**与**软件生命周期(IEC 62304)**文档?

### 常见疑问

| 疑问 | 解答 |
|---|---|
| 原型用普通 5V 电源行不行? | 桌面验证可以,但绝不能接真病人;产品必须换医疗电源 |
| 隔离了就一定安全吗? | 隔离是核心,但还要看漏电流、单一故障、EMC 等整体 |
| 共地和隔离矛盾吗? | 病人侧内部共地,病人侧与市电侧之间隔离——两不冲突 |
| 我能自己认证吗? | 合规涉及第三方测试与法规注册,远超个人 DIY 范围 |
| 那做原型还有意义吗? | 有——验证原理、练工程、出演示;只是别越过「非医疗器械」这条线 |

### 回望整个 H7:一次总演练

到这里,智能护理床项目走完了:CB1 定架构、CB2 听呼吸心跳、CB3 判翻身离床、CB4 测尿湿、CB5 保安全。回看会发现,它把前六个模块的知识几乎全用上了——**分压**读湿度电极、**ADC** 读称重与压力、**I²C** 接 IMU 与温湿度、**UART/CAN** 做通信、**电源与隔离**保安全,再加上**融合、报警、可靠性、合规**这些系统层面的工程思维。

更重要的是它教会一件事:**做一个真实系统,难的从来不是让某个传感器出个数,而是让整套东西在真实、嘈杂、会出故障、有人命关天的环境里可靠地工作。** 传感器是起点,系统工程与安全工程才是终点。这,就是从「会接线」到「能造系统」的分水岭。

> ⚠ **要点:** 医疗电子的第一原则是**患者安全**:把接触病人的部分与市电/大地**隔离**,把**漏电流**压到安全线以下——靠隔离电源、模拟/数字隔离器实现,病人侧自成悬浮的电气孤岛。用**LiFePO4 电池 + BMS**保证断电续航,用 **CAN** 组多床网。永远保持清醒:爱好级原型能学原理、能演示,但**距离合规医疗器械,还隔着一整套隔离、认证、EMC 与安全工程**(IEC 60601 系列)。
