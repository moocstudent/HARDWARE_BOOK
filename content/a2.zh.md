## 核心讲义

### 电机不能直接接引脚

电机是这门课里第一个「大功率」负载。它有两个特点让它绝不能像 LED 那样直接挂在单片机引脚上:

1. **电流大:** 即使小电机起步电流也可达几百毫安到几安,远超引脚上限(约 20–40 mA)。
2. **反电动势(back-EMF):** 电机本质是线圈,断电瞬间线圈会产生一个反向高压尖峰,足以击穿脆弱的芯片。

所以电机和单片机之间必须隔一个**驱动器件**:小电流场合用三极管/MOS 管,更常用现成的**驱动板**。

### H 桥:正反转的秘密

直流电机反接电压就反转。**H 桥**是四个开关组成的电路(形状像字母 H),通过控制哪两个开关闭合,就能让电流以两个方向流过电机,从而**正转 / 反转 / 刹车**。加上 PWM(见 IO3),还能**调速**。

常见驱动芯片 L298N、DRV8833 等本质都是 H 桥:

```cpp
// 假设 IN1/IN2 控制一路电机
digitalWrite(IN1, HIGH); digitalWrite(IN2, LOW);  // 正转
digitalWrite(IN1, LOW);  digitalWrite(IN2, HIGH); // 反转
digitalWrite(IN1, LOW);  digitalWrite(IN2, LOW);  // 停
analogWrite(ENA, 180);   // 用 PWM 调速(使能脚)
```

### 续流二极管:给反电动势一条出路

驱动感性负载(电机、继电器、电磁阀)时,要在负载两端反向并联一颗**续流二极管(flyback diode)**,断电瞬间让线圈电流从二极管安全泄放,而不是冲击开关器件。集成驱动板通常已内置。

### 步进电机:一步一步精确转

步进电机不是「通电就转」,而是**按脉冲一步一步转**——每收到一个脉冲,转过一个固定的小角度(如 1.8°/步)。这让它无需反馈就能**精确定位**,常用于 3D 打印机、CNC。它靠内部多组线圈**按顺序通电**产生「拉着转子走」的旋转磁场,一般配 ULN2003、A4988 等专用驱动板,由单片机发「步进 + 方向」信号。

> ⚠ **要点:** 电机 = 大电流 + 反电动势,永远经**驱动器**连接。直流电机用 **H 桥**实现正反转、用 **PWM** 调速,并加**续流二极管**保护;步进电机靠**分步脉冲**精确定位。给电机单独供电并与单片机**共地**。
