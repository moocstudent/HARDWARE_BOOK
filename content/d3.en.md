## Core Notes

### A digital pin only goes on/off — so how do you get "half bright"?

An output pin has just two levels, HIGH and LOW, so it seems it can only make an LED fully on or fully off. But if we switch it **very fast** and control *what fraction of the time it stays on*, the eye and the motor can't keep up with the rapid toggling and only feel an **average**. That's **PWM (Pulse-Width Modulation)**.

### Duty cycle sets the average voltage

The fraction of each cycle the pin is HIGH is the **duty cycle**:

- 0% duty → always LOW → average 0V → off
- 50% duty → half on, half off → average ~Vcc/2 → half bright
- 100% duty → always HIGH → average Vcc → full bright

> **Average voltage ≈ duty cycle × Vcc**

In the interactive figure, drag the duty and watch the square wave's high portion widen, the average line (dashed) rise with it, and the swatch brighten.

### analogWrite: a 0–255 scale

On Arduino, PWM is output with `analogWrite`, taking **0–255** (8-bit):

```cpp
analogWrite(9, 0);     // 0% duty    off
analogWrite(9, 64);    // ~25%       dim
analogWrite(9, 128);   // ~50%       half
analogWrite(9, 255);   // 100%       full
```

A "breathing" LED:

```cpp
void setup() { pinMode(9, OUTPUT); }
void loop() {
  for (int v = 0; v <= 255; v++) { analogWrite(9, v); delay(4); }
  for (int v = 255; v >= 0; v--) { analogWrite(9, v); delay(4); }
}
```

### Not every pin can PWM

`analogWrite` is true hardware PWM only on **specific pins**. On the Arduino Uno those are the ones marked `~`: **3, 5, 6, 9, 10, 11**. Calling `analogWrite` on other pins won't give smooth dimming.

### Frequency, flicker and smoothing

PWM **frequency** (how many switches per second) matters too. Too low and an LED visibly flickers, a motor judders or whines. The Uno defaults to ~490 Hz or ~980 Hz, plenty for dimming. For a genuinely smooth **analog voltage** (say to drive an op-amp), follow the PWM output with an RC low-pass filter (see CF4) to "flatten" the square wave into DC.

> ✦ **Key point:** PWM isn't true analog output; it uses "fast switching + duty cycle" to average an equivalent voltage over time. Dimming, speed control, generating an analog voltage — all PWM. Remember three things: duty cycle, which pins work, and frequency.
